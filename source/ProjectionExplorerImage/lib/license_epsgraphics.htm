EPS Graphics Overview

The EPS graphics library is a rerelease of the GNU GPL library that used to be available from jibble.org.

Java EPS Graphics Library started as a reworking of the source code of the EPS library that used to be available from jibble.org and was released under GNU GPL. Unfortunately the original authors decided to stop working on the GNU GPL code and moved to a commercial license.

We tracked down the last version we could find that was released under GPL and started from there. Currently our library is not much more than the latest release that was made by jible.org.

EPS Graphics Library version 1.0.0 released

We have released version 1.0.0 of the EPS Graphics Library.
This version is a rerelease of the code that was available under GNU GPL from http://jibble.org .

We have added support for an alternative color model besides the traditional RGB. The library support the CMYK color model that is often used for professional printing.
